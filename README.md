# ko-remove-aria-hidden-ada-compliance

Public release of the ko-remove-aria-hidden-ada-compliance WordPress plugin.

## Installation
1. Upload the plugin folder to /wp-content/plugins/
2. Activate in WordPress
3. Configure settings as needed

## Notes
- No secrets or API keys are stored in this repo
- Configure external services via admin or filters
